instuctions to run the program
1) run the makefile using `make all` command to build all the strstr.cpp numOccurrences.cpp anagram.cpp strstrx.cpp sources file's output OR run the makefile using `make [strstr|numOccurrences|anagram.cpp|strstrx ]` to build each source file individually.
2) run using `[strstr|numOccurrences|anagram.cpp|strstrx] needle_string haystack_string`
	a) for example strstrx chem alchemy
4) have a look at strstr.sh, numOccurrences.sh, anagram.sh and strstrx.sh on how to test effeciently.
	a) make sure that you do chmod +x for these bash scripts
